Drum Machine Web App

A fun and interactive Drum Machine built using HTML, CSS, and JavaScript. Each button generates a different drum sound, allowing users to create beats directly in the browser.

Project Overview

This project demonstrates frontend interactivity and sound integration in web development. Users can click on drum pads or use keyboard keys to play different drum sounds in real-time.

Features

Interactive drum pads for multiple sounds

Keyboard support for playing drums

Responsive and visually appealing design

Fun activity for music enthusiasts and beginners in JavaScript

Immediate audio feedback on user interaction

Technologies Used

HTML5 – Structure of the drum machine

CSS3 – Styling and layout

JavaScript – Audio handling and interactivity
